# Pet Telehealth System

A full-stack telehealth platform developed as a Final Year Project (FYP) to connect pet owners, veterinarians, and veterinary clinics through online healthcare services. The system provides appointment booking, medical record management, messaging, reminders, video consultations, and clinic administration in a single platform.

## Features

### Pet Owner

* Register and manage multiple pets
* View and update pet information
* Track pet health, weight, symptoms, and behaviour
* Book online or physical appointments
* View appointment schedules and reminders
* Chat with assigned veterinarians
* Access 24/7 veterinary support
* Join video consultations
* Export pet records as PDF documents
* Search and register with nearby veterinary clinics

### Veterinarian

* Manage assigned patients
* View and update pet medical records
* Conduct online consultations through video calls
* Chat with pet owners
* Create and manage message templates
* Use suggested reply recommendations
* View appointment schedules
* Manage availability for 24/7 support services

### Vet Admin

* Register and manage veterinarians
* Approve appointment requests
* Assign veterinarians to appointments
* Generate appointment slots
* Manage clinic schedules and operating hours
* Monitor clinic statistics and activity
* Manage patients registered under the clinic
